# 🏠 House Price Prediction using ANN (Regression)

This project demonstrates how to build an Artificial Neural Network (ANN) model for a regression problem using a house price prediction dataset.

## 📌 Features
- Data preprocessing (handling missing values, scaling)
- ANN model using TensorFlow/Keras
- Model evaluation using MSE and MAE
- Clean and modular code structure

## 🛠 Tech Stack
- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas & NumPy

## 📂 Project Structure
```
ann_house_price_project/
│
├── data/
│   └── housing.csv
├── src/
│   ├── preprocess.py
│   ├── model.py
│   └── train.py
├── requirements.txt
├── .gitignore
└── README.md
```

## 🚀 How to Run

### 1. Clone the repo
```bash
git clone https://github.com/your-username/ann-house-price.git
cd ann-house-price
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run training
```bash
python src/train.py
```

## 📊 Output
- Trained ANN model
- Evaluation metrics printed in console

## 🧠 Model Architecture
- Input Layer
- 2 Hidden Layers (ReLU)
- Output Layer (Linear)
